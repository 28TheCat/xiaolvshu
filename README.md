## 📖 Overview

**xiaolvshu** is a modular Java backend project built with Maven.

It is designed for:

- Learning backend architecture design
- Practicing Maven multi-module structure
- Building reusable common modules
- Standardizing API response and exception handling

This project follows clean structure principles and scalable design patterns.

---

## ✨ Features

- ✅ Unified response wrapper (`Response<T>`)
- ✅ Custom business exception abstraction
- ✅ Modular Maven structure
- ⏳ Global exception handling (planned)
- ⏳ Logging module (planned)
- ⏳ API standardization (planned)
